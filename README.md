
# Text Detection with OpenCV and EasyOCR

This project detects text in images using EasyOCR and OpenCV.

## Tools used

- Python
- OpenCV
- EasyOCR
- Matplotlib
- NumPy
- Google Colab

## Features

- Reads an image
- Detects text using EasyOCR
- Draws green bounding boxes around detected text
- Displays detected text on the image

## Sample Images

- Mercedes advertisement
  
## How to Run

1. Install dependencie
pip install easyocr opencv-python matplotlib numpy
2. Open the notebook.
3. Upload an image.
4. Run all cells.
